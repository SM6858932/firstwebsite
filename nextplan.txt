# 🌐 Complete Website Plan: Social Viral Platform

## 📋 Project Overview

A full-stack Next.js social platform where users can:

- Discover and join WhatsApp, Telegram, Instagram, and Facebook groups
- Watch viral Instagram reels uploaded by the admin
- Submit their own reels and groups for admin approval
- Engage with content through likes, comments, and shares

📱 API Routes
Endpoint	Method	Description
/api/reels	GET	Fetch all approved reels
/api/reels	POST	Admin uploads new reel
/api/reels/:id	PUT	Update reel (likes, comments, shares)
/api/reels/:id	DELETE	Delete reel
/api/groups	GET	Fetch all approved groups
/api/groups	POST	User submits new group
/api/groups/:id	PUT	Admin updates group
/api/groups/:id	DELETE	Delete group
/api/submissions	GET	Admin fetches pending submissions
/api/submissions/:id	PUT	Admin approves/rejects submission
/api/stats	GET	Admin dashboard statistics

🔗 Important Implementation Notes
Option A: Manual Upload (Recommended)
text
✅ Advantages:
  - Fast to implement
  - No API dependency
  - No legal issues
  - Works forever
  - Full control over data

❌ Disadvantages:
  - Manual updates required
  - Engagement metrics may become outdated
Implementation:

Admin manually enters likes, comments, shares

Thumbnail uploaded via form

Reel URL links to Instagram for viewing

Option B: Meta Graph API (Advanced)
text
✅ Advantages:
  - Automatic engagement updates
  - Real-time metrics
  - Official Meta integration

❌ Disadvantages:
  - Facebook Developer Account required
  - Instagram Business Account required
  - Access Token with specific permissions
  - Rate limits
  - Only supports approved use cases
Requirements:

Facebook Developer Account

Instagram Business Account

Access Token with instagram_basic and pages_read_engagement permissions
