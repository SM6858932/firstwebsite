# Launch Checklist

## Recommended production domain
- Use one primary domain for all public-facing launch work.
- For Google AdSense, Search Console, analytics, and social sharing, use the same canonical domain consistently.
- If both Vercel preview domains are active, choose the one that will remain the primary production URL and update all metadata and verification settings to match it.

## Suggested setup
- Verify the production domain in Google Search Console.
- Add the same domain to Google Analytics and AdSense.
- Keep canonical tags and Open Graph URLs aligned with the final production URL.
- Replace the placeholder OneSignal app ID with the real app ID before public launch.
